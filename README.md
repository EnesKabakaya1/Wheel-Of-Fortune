
# Wheel Of Fortune

A wheel of fortune developed for use in many needs using html, css, js.

## Run Locally

Clone the project

```bash
  git clone https://github.com/EnesKabakaya1/Wheel-Of-Fortune
```

Go to the project directory

```bash
  cd my-project
```
